Instruções para o Desenvolvimento em Equipe no Github

Passo 1 - Ao acessar o Git Bash de dentro da pasta do projeto, entre no branch master
> git checkout master

Passo 2 - Atualize a pasta do seu computador com o repositório online (para você ter tudo o que pode ter sido enviado por outro integrante da equipe)
> git pull origin master

Passo 3 - Crie a sua Branch de modificação a partir da versão master do repositório
> git checkout -b Nome_da_Branch master

Passo 4 - Feche o Git Bash e faça o seu trabalho, abrindo a Unit pela pasta do projeto (onde você abriu o Git Bash). Ao terminar, salve o projeto na Unit e reabra o Git Bash da pasta do projeto.

Passo 5 - Acesse a branch master novamente
> git checkout master
